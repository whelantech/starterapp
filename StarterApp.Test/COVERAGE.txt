SonarCloud coverage (OpenCover, reliable with Sonar C# analyzer)

- CI generates OpenCover via Coverlet: test step uses `coverlet.runsettings` + `--collect:"XPlat Code Coverage"`.
- Report path: `**/coverage.opencover.xml` (under `StarterApp.Test/TestResults/.../`) — see `sonar-project.properties` → `sonar.cs.opencover.reportsPaths`.
- Local: `dotnet test StarterApp.sln -f net10.0 --settings coverlet.runsettings --collect:"XPlat Code Coverage"` then open the Attachment path printed at the end of the test run.
- Scanner reads `sonar-project.properties` at repo root; `sonar.sources` / `sonar.tests` use `StarterApp` names (not `RentalApp`).
